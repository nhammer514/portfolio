SVG

Elements and graphics on webpages are commonly made using SVG. One of the most useful features of SVG is animation. 


<circle cx="50%" cy="125" r="100" stroke="black" stroke-width="1" fill="blue">
    <animate attributeName="r" values="100;125;100" dur="3s" repeatCount="indefinite"/>
</circle>

JavaScript can be implemented into SVG elements.
