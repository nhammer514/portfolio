DOMContentLoaded Event

Event Listeners are functions that are listening for input ( or events ) coming from the HTML page.


In the code below, we created an event listener. This code block is also implemented into this page.

window.addEventListener('DOMContentLoaded',init,false);
function init(){
    alert("Hello world!")
};

Click Event

Event Listeners can be used for a variety of different features. For instance, event listeners can be used for buttons.



The code below reflects the functionality of the button. We are grabbing all buttons on this page, assigning an event listener for each button, and triggering unique functions when they are clicked on.

window.addEventListener('DOMContentLoaded',init,false);
function init(){
    const exampleButton = document.getElementById('testExample');
    exampleButton.addEventListener('click',buttonClicked,false);
}
function buttonClicked(){
    alert("You clicked me!")
}

Query Selector & Loops

There are many ways to iterate through elements and arrays.


Outside of using a "for loop" function, you can use the "forEach()" function, in conjunction with the arrow function syntax.



Keep in mind, the button in this example is not using an eventListener. Instead, we are calling the grabNumOfElement function inside the HTML (view source). 


function grabNumOfElement(tagName){
    let numOfElement = 0
    document.querySelectorAll(tagName).forEach( header => {
      numOfElement += 1;
    })
    alert("Number of " + tagName + " elements: " + numOfElement);
}
